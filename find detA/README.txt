Алгоритм разработан для восстановления перестановки P по таблице инверсий W.
В этом алгоритме нужно последовательно просматривать компоненты таблицы инверсий Wi
справа налево от i=n до i=1, чтобы получить из их индексов список L, который показывает
относительное расположение элементов pi искомой перестановки со значениями от n до 1, соответственно.
Сначала в пустой список включается элемент со значением n. На каждом следующем шаге очередной элемент со значением i
располагается так, чтобы в списке перед ним оказалось Wi элементов, значения которых больше, чем i. После размещения
последнего элемента со значением 1 в списке получается искомая перестановка, которая соответствует заданной таблице инверсий.
